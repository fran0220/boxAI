## Do not translate

- Provider names: Claude, OpenAI, Gemini, Grok, Anthropic, Vertex, Stripe, Airwallex, WeChat, DingTalk, LinuxDo
- Error reason codes: `USER_NOT_ACTIVE`, `PAYMENT_*`, etc.
- HTTP paths, env vars (`BATCH_IMAGE_VERTEX_MANAGED_GCS_BUCKET`), technical IDs
- Brand mark `BoxAI`
